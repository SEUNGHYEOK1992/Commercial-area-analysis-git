# 맡은기능(관리자페이지)
관리자페이지를 담당하였습니다.
관리자 페이지를 만들 때 페이징 처리를 비동기 방식이 아닌 동기화 방식으로 시도를 했습니다.
동기화 방식으로 진행하면서 중간중간 오류가 나서 수업 시간 때 배웠던 자료를 참고하여 어디서부터 오류가 난 것인지 차근차근 확인했습니다.
원인은 폼 방식으로 바꾸지 않았던 것이 원인이었고 폼 방식으로 바꾼 뒤에 실행했을 때 페이징 구현이 성공적으로 되었습니다.
